# Apifox / 接口回归测试清单

> 无云阶段可在 **dev（localhost:8080）** 执行；上线前须在 **staging** 复跑并归档。  
> 导入 OpenAPI：见 [README.md](./README.md)

## 环境变量

| 变量 | dev 示例 |
|------|----------|
| `baseUrl` | `http://localhost:8080` |
| `memberToken` | 学号登录或 wx-login 返回 |
| `adminToken` | `POST /api/v1/admin/auth/login` |

## P0 必测（每次发版）

### 鉴权

| 方法 | 路径 | 预期 |
|------|------|------|
| POST | `/api/v1/auth/account-login` | 200 + token |
| POST | `/api/v1/auth/wx-login` | dev-mode 200 |
| GET | `/api/v1/profile` | 带 token 200；无 token 401 |

### 内容读接口

| 方法 | 路径 | 预期 |
|------|------|------|
| GET | `/api/v1/news` | 200 列表 |
| GET | `/api/v1/news/{id}` | 200 详情含 viewCount |
| GET | `/api/v1/halls` | 200 ≥11 条 |
| GET | `/api/v1/halls/{id}` | 200 含 vrUrl/vrReady |
| GET | `/api/v1/courses` | 200 |
| GET | `/api/v1/home/recommends` | 200 |

### 报名（边界）

| 方法 | 路径 | 预期 |
|------|------|------|
| POST | `/api/v1/activities/{id}/enroll` | 正常 200 |
| POST | 同上（满员活动） | 业务错误码，非 500 |
| POST | 同上（重复报名） | 明确提示 |

### AI

| 方法 | 路径 | 预期 |
|------|------|------|
| GET | `/api/v1/ai/chat/quota` | 200 含 remaining |
| POST | `/api/v1/ai/chat/sessions` | 200 sessionId |
| POST | `/api/v1/ai/chat/sessions/{id}/messages` | 200 含 content |

### 管理端

| 方法 | 路径 | 预期 |
|------|------|------|
| POST | `/api/v1/admin/auth/login` | 200 |
| GET | `/api/v1/admin/news` | 200 |
| POST | `/api/v1/admin/news` | 200 创建 |
| GET | `/api/v1/admin/stats/overview` | 200 |

## P1 建议测

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/news/{id}/like` | 点赞切换 |
| GET | `/api/v1/search` | 全局搜索 |
| GET | `/api/v1/messages` | 消息列表 |
| POST | `/api/v1/admin/knowledge/docs` | 知识库录入 |
