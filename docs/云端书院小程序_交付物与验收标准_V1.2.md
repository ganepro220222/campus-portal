## 二、移动端模块验收标准

### 2.1 登录与身份

| 验收项 | 标准 |
|---|---|
| 微信授权登录 | 点击微信授权，返回有效 token，可访问需登录接口 |
| 学号/账号登录 | 正确账号可登录；错误密码返回明确提示；连续 5 次错误锁定 5 分钟 |
| 游客浏览 | 未登录可访问新闻/展馆/文创/课程列表及详情；点击收藏/报名跳转登录页 |
| 个人信息绑定 | 首次登录引导填写；报名表单自动预填已绑定信息 |
| 退出登录 | 退出后 token 失效，重访需重新登录 |

### 2.2 首页与核心入口

| 验收项 | 标准 |
|---|---|
| Banner 轮播 | 后台配置图片和跳转链接，小程序自动轮播，点击跳转正确 |
| 功能入口矩阵 | 后台配置入口，点击跳转对应页面，禁用的入口不展示 |
| 推荐内容 | 展示 3 条新闻和 2 门课程，与后台配置一致 |
| 公告通知条 | 后台发布公告后首页展示；过期公告自动隐藏 |
| 全局搜索 | 输入关键词返回新闻/展馆/文创/课程/资源的跨模块结果 |

### 2.3 新闻资讯

| 验收项 | 标准 |
|---|---|
| 分类浏览 | 按分类标签筛选，下拉加载更多 |
| 详情展示 | 图文混排正常；视频可播放；浏览量在 5 分钟内同步增加 |
| 点赞/收藏 | 点赞取消实时更新数量；收藏后个人中心可查看；再次点击取消 |
| 分享 | 分享至好友/朋友圈可生成分享卡片，链接可打开对应详情页 |
| 智能推荐 | 详情页底部展示同分类最近 5 条，不含当前文章 |

### 2.4 线上展馆

| 验收项 | 标准 |
|---|---|
| 展馆列表 | 11 个展馆正常展示，按后台排序 |
| 详情浏览 | 分区展示；图片支持点击放大和手势缩放 |
| 语音讲解 | 点击播放/暂停；切换展馆时上一个停止；退出页面时释放资源 |
| 沉浸式浏览 | 连续滚动，章节分区，底部进度感知 |
| 海报生成 | Canvas 合成展馆封面+名称+校徽，可保存到手机相册 |

### 2.5 文创展示

| 验收项 | 标准 |
|---|---|
| 高清图鉴赏 | 多角度图片正常展示，支持点击放大 |
| 语言切换 | 中英文版本切换正常，内容独立 |
| 立体展示/降级 | 有 GLB 模型时 xr-frame 渲染；无模型时多角度图片轮播 |
| 购买咨询 | 电话一键拨打；微信号一键复制并提示"已复制" |

### 2.6 课程中心

| 验收项 | 标准 |
|---|---|
| 视频播放 | 可播放/暂停/拖动进度；封面图（poster）正确展示，无黑屏 |
| 字幕 | 字幕可开关；字幕与视频进度同步 |
| 学习进度 | 每 20 秒上报一次；退出后重进，视频从上次位置开始 |
| 完成统计 | 播放 ≥ 90% 标记为已完成，触发积分 |
| 配套资源 | 课程详情页可直接下载/预览配套资源 |

### 2.7 资源下载

| 验收项 | 标准 |
|---|---|
| 分类浏览 | 按课件/视频/音频分类展示，支持格式筛选 |
| 预览 | PDF 调用微信文件预览；音频直接播放；视频跳转播放页 |
| 下载记录 | 点击下载后，个人中心可查看下载记录，含文件名和时间 |

### 2.8 活动报名

| 验收项 | 标准 |
|---|---|
| 报名流程 | 填写信息并提交，报名成功后展示凭证码和二维码 |
| 防重复 | 同一用户重复报名提示"您已报名该活动" |
| 名额控制 | 名额满时提示"名额已满"，不可继续报名 |
| 并发安全 | 50 并发同时报名，最终报名人数不超过名额上限 |
| 审核模式名额 | 开启审核的活动：提交报名即占名额（pending 状态）；管理员拒绝或用户取消时释放名额 |
| 取消报名 | 取消后可重新报名（活动报名时间内）；已满的名额释放一个 |
| 活动取消 | 管理员取消活动后，待审/已通过报名同步取消并释放名额；学员收到站内消息 |
| 微信提醒 | 有授权的用户收到报名成功微信通知；无授权的用户消息中心可查看 |

### 2.9 学院矩阵

| 验收项 | 标准 |
|---|---|
| 入口展示 | 后台配置的学院入口按排序展示 |
| 内容呈现 | 至少一种对接方式（手动录入/H5/接口/跳转）正常工作 |
| 跳转方式 | 跳转前出现微信系统确认弹窗（平台行为，正常现象） |

### 2.10 个人中心

| 验收项 | 标准 |
|---|---|
| 收藏汇总 | 新闻/展馆/文创/课程/资源的收藏统一展示 |
| 报名记录 | 展示**有效报名**及状态（待审核/已通过/已拒绝）；已取消的报名不在列表展示，活动取消等通过消息中心告知 |
| 下载记录 | 展示历史下载，可再次下载 |
| 学习足迹 | 时间轴展示最近 30 天的浏览/学习/报名记录 |
| 积分徽章 | 积分数量准确；已获徽章高亮展示；未获徽章显示获取条件 |

### 2.11 AI 文化问答

| 验收项 | 标准 |
|---|---|
| 知识回答 | 上传书院文化资料后，相关问题可基于资料内容作答 |
| 内容安全 | 违规问题被拦截，返回友好提示 |
| 次数限制 | 每用户每日 20 次上限，超限提示次日再来 |
| 会话历史 | 可查看历史会话记录 |

## 三、管理后台验收标准

| 模块 | 验收标准 |
|---|---|
| 登录权限 | 管理员登录成功；不同角色访问权限边界正确；越权返回 403 |
| 新闻管理 | 发布/下架/置顶/定时发布均正常；AI 辅助返回润色结果 |
| 展馆管理 | 多图上传/排序/语音上传正常；下架后小程序不可见 |
| 文创管理 | 中英文内容独立维护；联系方式配置后小程序同步 |
| 课程管理 | 视频上传后异步转码；字幕生成状态可查；AI 简介辅助可用 |
| 活动管理 | 可新建/发布/取消活动；报名名单可筛选/审核/导出 |
| 资源管理 | 上传/下架/统计下载次数均正常 |
| AI 知识库 | 文档上传后处理状态变为 ready，问答效果更新 |
| 用户管理 | 禁用后用户无法登录；积分规则修改后生效 |
| 统计看板 | 数据与数据库一致；图表正常渲染；Excel 可正常下载 |
| 操作日志 | 所有后台修改操作有日志记录，可按时间和操作人查询 |
| Banner/公告 | 配置后小程序 5 分钟内（缓存 TTL）同步展示 |

## 五、性能基准

| 指标 | 目标值 | 测试方法 |
|---|---|---|
| 接口响应（缓存命中） | P95 < 50ms | ab 压测热点接口 |
| 接口响应（数据库查询） | P95 < 300ms | ab 压测冷接口 |
| 小程序首屏加载 | < 2s（4G 网络）| 真机测试 |
| 视频首帧加载 | < 3s（4G 网络）| 真机测试 |
| 并发浏览 | 200 并发下接口 P95 < 500ms | JMeter 压测 |
| 并发报名 | 50 并发下报名数量不超额 | ab -n 200 -c 50 |

## 六、安全验收清单

- [ ] 所有接口强制 HTTPS
- [ ] 未登录访问需登录接口返回 401
- [ ] 管理员越权访问返回 403
- [ ] 文件上传格式白名单限制有效（上传 .php 等被拒绝）
- [ ] OSS 文件需签名 URL 才能访问（直接访问 OSS 原始地址返回 403）
- [ ] AI 问答输入安全审核有效
- [ ] AI 生成内容安全审核有效
- [ ] 数据库不存明文密码（BCrypt 加密验证）
- [ ] 操作日志记录完整
