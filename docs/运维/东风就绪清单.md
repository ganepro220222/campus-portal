# 东风就绪清单

> 目标：**本地代码/脚本/文档闭环**，云资源与甲方资料到位后「填表 → 跑脚本 → 部署」即可。  
> 最后更新：2026-07-13 · 基线 `04c1701` 之后 P0–P4 落地

## 1. 发布前本地预检（无需云）

```bash
npm run preflight:local
# 串联：test:deploy-env + check:deploy-env + test:mock-guard + test:sync-navigate-appids
```

云资源与 `.env` 填好后：

```bash
npm run check:release-all -- --env .env
cd backend && mvn test          # 需 Java 17
cd admin && npm run typecheck && npm run build
```

## 2. 代码就绪 vs 东风待填

| 模块 | 代码状态 | 东风（外部依赖） | 填哪里 / 怎么做 |
|------|----------|------------------|-----------------|
| 后端 API + 管理端 | ✅ 就绪 | 域名 HTTPS、RDS、Redis | `.env.prod.template` → `.env` |
| 微信小程序登录 | ✅ code2session | WX AppID / Secret | `.env` `WX_APPID` `WX_SECRET` |
| 订阅消息（报名） | ✅ 骨架 | 模板 ID（**不含活动提醒**） | `WX_TMPL_ENROLL_*` |
| 关联小程序 jump | ✅ 首页横滑 + 个人中心 + 列表 | 兄弟小程序 AppID 清单 | `navigate-appids.json` → `sync-navigate-appids.js` |
| 关联 api_sync | ⏳ **预留可见** | 对方 API 文档 + 密钥 | 管理端可存；小程序提示「预留待对接」 |
| 海报小程序码 | ✅ getwxacode + dev 降级 | 生产 WX 凭证 | 同微信 AppID；`GET /miniapp/wxacode` |
| 课程字幕 ASR | ✅ 阿里云 Provider + 轮询 Job | ASR Key / AppKey | `ASR_*` 环境变量 |
| AI 问答 | ✅ 关键词 MVP | 可选智谱 Key | `AI_API_KEY`（非阻塞） |
| OSS 媒体 | ✅ 主链 | Bucket + CDN | `OSS_*` |
| staging 部署/压测 | ⏳ | 云资源 | 部署手册 §5.3 |

## 3. 东风到位后 30 分钟流程

```bash
# 1. 生产配置
cp .env.prod.template .env
cp miniapp/config/env.prod.template.js miniapp/config/env.js   # ENV=prod + 域名
cp miniapp/config/navigate-appids.template.json miniapp/config/navigate-appids.json

# 2. 编辑 .env / env.js / navigate-appids.json 填入真实值

# 3. 小程序跳转白名单
node scripts/sync-navigate-appids.js

# 4. 预检
npm run preflight:local
npm run check:release-all -- --env .env

# 5. 部署 — 见 docs/运维/部署手册_V1.0.md §5.3
```

## 4. 关联小程序四种方式（管理端均可配置）

| 方式 | 小程序端 | 未对接时行为 |
|------|----------|--------------|
| manual | 弹窗简介 | 正常展示 |
| jump | `navigateToMiniProgram` | AppID 错/未白名单 → Toast 跳转失败 |
| embed_h5 | web-view | 无 URL → Toast |
| api_sync（预留） | Modal 说明预留 | **不隐藏**；待甲方 API 后实现拉取 |

**说明：** Swagger 描述的是「别人调我们」；`api_sync` 是「我们调别人」，方向相反。

## 5. V1.0 明确不做

- `activity_remind` 活动开始提醒订阅消息
- `api_sync` 真实拉取逻辑（仅预留配置与提示）
- AI 向量检索（维持关键词 MVP）
