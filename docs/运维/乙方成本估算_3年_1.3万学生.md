## 一、说明与边界

### 1.1 本文估算什么

本文**仅估算可向校方说明、可发票/账单核对的第三方与云资源费用**：

| 类别 | 是否纳入 |
|------|----------|
| 云服务器、云数据库、Redis、OSS、CDN | ✅ |
| 微信小程序认证、域名、SSL（免费证书） | ✅ |
| 上线首年预发（staging）环境机费 | ✅ |
| 语音转写（课程字幕，阿里云 ISI 按量） | ✅（一次性，金额很小） |
| **小程序 AI 问答、管理端文案辅助**（大模型推理按量） | ✅ |
| **知识库向量化**（上线首年建库） | ✅（一次性，金额很小） |
| **研发人力 / 外包人天** | ❌ **不纳入** |
| **开发阶段使用的编程辅助工具** | ❌ **不纳入，本文不提** |
| 校方内容素材制作、等保测评 | ❌ |

### 1.3 估算口径

- 金额单位：**人民币（元）**。
- **基础生产（A）** 与 **稳妥生产（B）** 两档；小程序 AI 按 **低 / 中 / 高** 三档用量估算（默认取**中档**汇总）。
- **第 2、3 年**按云厂商**续费价**（通常较首年活动价上浮 **15%–30%**）及 DAU 温和增长估算。
- 若学校通过阿里云「[云工开物](https://university.aliyun.com/)」等计划获得折扣，云资源项可整体下调（见 §六）。

### 3.1 首年预发环境（staging）

| 项 | 金额（元） | 说明 |
|----|-----------|------|
| ECS 2C4G（staging） | **400** | 仅首年需要独立预发；**第 2 年起可关停或合并** |

### 3.2 语音转写费用（推导，一次性）

假设首年入库 **50 门课**，平均每门 **25 分钟**：

```
总时长 ≈ 50 × 25/60 ≈ 20.8 小时
闲时版 ≈ 1 元/小时（阿里云 ISI 公开价）
费用 ≈ 21 元；含重跑取 200 元预算
```

### 3.3 知识库向量化（一次性，上线建库）

依据《技术实施方案 V1.2》§6.6，文化问答采用 MySQL 存 chunk + 应用层检索；若使用百炼 **text-embedding** 类模型做建库向量化：

```
假设首批入库资料 ≈ 80 篇 × 平均 2,000 Token ≈ 16 万 Token
embedding 公开价量级 ≈ 0.1–0.5 元/百万 Token
一次性向量化 ≈ 0.02–0.08 元（纯计算极低）
```

实际账单主要来自**调试重嵌入、增补资料**；预算取 **100 元**。

### 4.2 小程序与管理端 AI 能力（按年，产品按量）

对应验收 §2.11（文化问答）、§三（管理端 AI 文案辅助）。微信 `msgSecCheck` 内容安全在小程序体系内**通常无额外按次费**（以微信当期政策为准）。

**单次问答 Token 估算**（RAG：Top-5 段落 + 生成回答）：

```
输入约 3,000 Token + 输出约 800 Token
单次推理 ≈ 0.3×0.003 + 0.6×0.0008 ≈ 0.0014 元/次（qwen-turbo 华北2 公开价）
```

**日活与使用率**（与《资源配置清单》常态 DAU 1,300 对齐；《技术实施方案》上限 20 次/人/日）：

| 档位 | 假设 | 年调用量（约） | Year 1 | Year 2 | Year 3 |
|------|------|----------------|--------|--------|--------|
| **低** | 10% DAU，人均 3 次/日 | 14 万 | **200** | **250** | **300** |
| **中（推荐）** | 15% DAU，人均 5 次/日 | 36 万 | **500** | **600** | **700** |
| **高** | 25% DAU，人均 8 次/日 | 119 万 | **1,700** | **2,000** | **2,300** |

管理端文案辅助（润色/摘要）已含在上表调用量中，按全站总调用 **<5%** 计，不单独拆行。

> **控费手段（方案已设计）：** 每用户每日 20 次上限、相同问题 24h 缓存、后台开关与额度预警（V1.2 §6.6、V1.1 §9.6）。  
> 理论极端上限（全员打满 20 次/日）≈ 948 万次/年，**不作为预算基线**。

## 六、三年费用结构（方案 A + AI 中档）

| 费项 | 三年合计 | 占比 |
|------|----------|------|
| CDN 流量 | ≈ 1,950 | **28%** |
| ECS 应用服务器 | ≈ 3,900 | **27%** |
| RDS 数据库 | ≈ 1,350 | **9%** |
| Redis | ≈ 1,400 | **10%** |
| 微信认证 | 900 | **6%** |
| OSS 存储与请求 | ≈ 1,130 | **8%** |
| **小程序 AI 问答等** | **≈ 1,800** | **12%** |
| 域名 + 语音转写 + 向量化 + staging | ≈ 1,180 | **8%** |
| **合计** | **≈ 14,610** | 100% |

**结论：** 在 1.3 万学生、文化类小程序常态 DAU 下，**三年账单化支出约 1.5 万–2.0 万元**；**CDN、ECS 仍占约五成**，**小程序 AI 产品按量约占一成**（中档），可通过限次与缓存继续压低。

## 七、可选减免与风险

| 项 | 影响 |
|----|------|
| **云工开物 / 高校折扣** | 云资源三年可省约 **4,000–6,000 元**（以审核为准） |
| **首年活动价续费上浮** | Year 2 起账单可能 +20%～30%，已计入上表 |
| **DAU 超预期** | CDN 线性上涨（约 +300–600 元/年/每 500 DAU）；AI 调用同步上升，建议收紧日限或加大缓存 |
| **校方大量上传视频** | OSS 存储与 CDN 需重算；**>500GB** 素材建议单独评估 |
| **等保测评** | 若校方单独立项 **1–5 万元/次**，不含在上表 |
