# 云端书院 · 甲方配合事项清单

> 贵州交通职业大学 ·「马院+书院」协同育人项目  
> 用途：明确 **校方需提供/确认** 的事项，避免上线前扯皮。  
> 乙方负责采购云资源、开发与部署；校方配合授权、资料与审核。  
> **双轨勾选清单（与校方 Excel 表对应）：** [上线分工checklist_V1.0.md](./上线分工checklist_V1.0.md)

## 一、上线前必须（P0）

| 序号 | 配合项 | 校方动作 | 不提供的影响 |
|------|--------|----------|--------------|
| 1 | 微信小程序主体与类目 | 提供主体资质、配合类目审核 | 无法提审上线 |
| 2 | 生产 AppID / Secret | 公众平台获取后 **线下安全交接** | 真机微信登录不可用 |
| 3 | 备案域名 | 提供域名并完成 ICP 备案 | 无法配置 HTTPS 合法域名 |
| 4 | DNS 解析 | 按乙方提供的记录配置 API/Admin/CDN 子域 | 外网无法访问 |
| 5 | 微信服务器域名 | 公众平台添加 request/upload/download 合法域名 | 小程序无法请求 API |
| 6 | 微信业务域名 | 添加 VR 外链域名（如 `eqvrar.com`、`bafang720.com`、`720yun.com` 等） | 展馆 VR web-view 打不开 |

## 二、试运行期必须（P1）

| 序号 | 配合项 | 校方动作 | 不提供的影响 |
|------|--------|----------|--------------|
| 7 | 订阅消息模板 ID | 在公众平台申请报名成功等模板，提供模板 ID | 仅站内消息，无微信推送 |
| 8 | 兄弟小程序清单 | 提供 AppID、path、名称、图标、入口位置 | 「关联小程序」无法正式配置 |
| 9 | 管理后台账号名单 | 确认超管/编辑/审核角色与初始账号 | 生产只能用默认 dev 账号（禁止） |
| 10 | 运营内容素材 | 新闻、展馆图文、课程视频、知识库资料 | 平台空壳，AI 无法答好 |
| 11 | 隐私政策 / 用户协议 | 法务审定正文（可用乙方占位稿为基础） | 微信审核可能卡项 |
| 12 | AI API Key（若指定校方平台） | 提供学校主体申请的 Key | 使用乙方 Key 或兜底模式 |

## 三、可选 / 持续（P2）

| 序号 | 配合项 | 说明 |
|------|--------|------|
| 13 | 品牌素材 | Logo、校徽高清图、首页 Banner 设计稿 |
| 14 | 统计口径确认 | DAU/PV/报表对外解释口径 |
| 15 | 兄弟学院 API 文档 | 若需接口同步学院内容（P2 能力） |
| 16 | 鸿蒙机型加测 | 非验收硬性项，可选 1 台华为机体验版抽测 |

## 四、乙方负责（无需校方采购，但需授权时配合）

- ECS / RDS / Redis / OSS / CDN 开通与配置
- SSL 证书申请与续期（DNS 验证需校方解析配合）
- 源代码、部署、备份恢复、监控告警实施
- 微信小程序代注册/认证办理（需校方主体授权）
